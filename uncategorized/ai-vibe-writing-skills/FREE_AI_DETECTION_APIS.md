# 免费 AI 写作检测 API 推荐指南

本文档按“免费额度、易用性、中文支持”维度，整理了适合集成到 `content-review-agent` 的免费 AI 写作检测与查重 API，适合个人开发者与小项目使用。

## 一、Copyleaks AI Detector API（推荐）
*   **免费额度**：新用户每月 10,000 免费字符（约 2500 字），可检测 AI 生成 + 抄袭
*   **支持**：30+ 语言、ChatGPT/Gemini/Claude/DeepSeek 等，句子级标记
*   **获取**：注册 → 进入 API 面板 → 生成 API Key
*   **接口**：`https://api.copyleaks.com/v3/scans/submit/file`
*   **适合**：需要 AI + 抄袭双检测、多语言场景

## 二、Sapling AI Content Detector API
*   **免费额度**：每月 1,000 次免费检测（单篇 ≤ 5000 字符）
*   **优势**：中文友好、句子级高亮、误报率低（<3%）
*   **获取**：注册 → 开发者中心 → 申请 API Key
*   **适合**：写作工具、内容审核、中文内容检测

## 三、Dechecker AI Checker API（开源免费）
*   **免费**：完全开源、无额度限制、可本地部署
*   **功能**：AI 概率评分、句子级分析、支持多模型
*   **获取**：GitHub 下载 → 自行部署 → 本地调用
*   **适合**：技术团队、隐私敏感、需要私有化部署

## 四、腾讯朱雀 AI 检测助手（中文优先）
*   **免费额度**：登录后每日 20 次文本检测（单篇不限长）
*   **优势**：深度优化中文、支持豆包/文心/DeepSeek 等国产模型
*   **获取**：腾讯云账号 → 朱雀 AI 检测 → 申请 API 权限
*   **适合**：中文内容、教育/学术场景、国产 AI 检测

## 五、Hugging Face 开源检测模型（免费调用）
*   **免费**：基于开源模型（如 roberta-base-openai-detector），无官方额度限制（受平台限流）
*   **方式**：用 Transformers 库本地运行，或通过 Hugging Face Inference API 调用
*   **适合**：技术开发者、自定义检测逻辑、低成本批量检测

## 六、Phrasly AI Detector（完全免费，无 API 但可爬虫）
*   **免费**：无限次、无限字符、不存储内容
*   **限制**：无官方 API，需通过网页爬虫/模拟请求接入
*   **适合**：个人临时检测、非商业小项目
